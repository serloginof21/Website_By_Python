# Тестовый сайт на Python - Система оценок игр
🎮 GameReview — Сайт с обзорами видеоигр
GameReview — это полноценное веб-приложение для публикации обзоров видеоигр. Платформа позволяет создавать, редактировать, удалять статьи, оценивать игры, отслеживать популярность контента и удобно искать нужные обзоры.

✨ Возможности проекта:
- 📝 Управление статьями
- ➕ Создание статей — добавление новых обзоров с названием, описанием и полным текстом
- ✏️ Редактирование — возможность изменять любую статью
- 🗑️ Удаление — удаление статей с подтверждением
- 📅 Автоматическая дата — каждая статья получает дату создания

-------------------------------------------------

⭐ Система оценок:
- Оценка игр от 1 до 10
- Визуальное отображение звездного рейтинга
- Цветной бейдж с оценкой на карточке статьи
- Сортировка статей по оценке

-------------------------------------------------

👁️ Счетчик просмотров:
- Автоматический подсчет просмотров каждой статьи
- Отображение количества просмотров в карточках
- Блок "Популярные обзоры" на главной странице
- Боковая панель с популярными статьями на странице обзора
- Сортировка статей по популярности

-------------------------------------------------

🔍 Поиск и фильтрация:
- Поиск статей по названию и описанию
Три варианта сортировки:
- 📅 По дате (новые сверху)
- ⭐ По оценке (высокие сверху)
- 👁️ По популярности (много просмотров)
